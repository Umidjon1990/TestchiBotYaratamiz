Example of Structured Mode (`input_mode: "0"`)
curl -X POST "https://lahajati.ai/api/v1/text-to-speech-absolute-control" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-H "Accept: audio/mpeg" \
-d '{
  "text": "هذا اختبار للوضع المنظم.",
  "id_voice": "ac_voice_xyz789",
  "input_mode": "0",
  "performance_id": "perf_1",
  "dialect_id": "dialect_10"
}' \
--output structured_output.mp3
